# AI-smart_health_monitoring

🔄 System Workflow

1️⃣ Data Collection

Sensor data collected from wearable devices.

2️⃣ Data Preprocessing

Cleaning missing values

Normalization

Window segmentation

Feature extraction

3️⃣ Model Training

CNN

LSTM

DNN

DBN

ELM

4️⃣ Deep Ensemble Learning

Combine predictions from all models.

5️⃣ Health Monitoring Dashboard

Displays predictions and live sensor readings.

📈 Performance Metrics

The model performance is evaluated using:

Accuracy

Precision

Sensitivity (Recall)

Specificity

F1-Score

Matthews Correlation Coefficient (MCC)

📊 Example Output

The system predicts activities such as:

Walking

Sitting

Standing

Running

Laying

and visualizes the results in a health monitoring dashboard.
